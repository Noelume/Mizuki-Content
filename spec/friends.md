## 友链说明

欢迎交换友链，长期维护、内容正常更新的网站都可以申请。  
如果你已经添加了本站链接，可以通过发送邮件申请。

## 申请前请确认

- 网站可正常访问，非空站、非采集站。
- 内容方向清晰，尽量保持原创或高质量转载。
- 页面加载速度正常，移动端可访问。
- 已添加本站链接（建议先友后链）。

## 申请格式（可直接复制）

```yaml
名称: 你的站点名
链接: https://example.com
头像: 你的头像图片链接 (任意图床或 GitHub 头像皆可)
描述: 一句话介绍你的网站
标签: [技术, 生活]
```

## 申请方式

如果你已经添加了本站链接，请将你的站点信息（按照上述申请格式）发送邮件至：

📧 **`Noelumeblog@outlook.com`**

## 备注

友链会不定期巡检。若站点长期无法访问或内容异常，可能会暂时下链，恢复后可再次联系补链。
